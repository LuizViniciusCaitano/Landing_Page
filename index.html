Programação mobile

desenvolvimento de aplicativos ou software projetados para dispositivos móveis, como smartphones e tablets. Ela envolve a criação de aplicativos nativos, híbridos ou baseados na web que podem ser executados em sistemas operacionais móveis, como Android e iOS.

Front-end

parte de um sistema de software ou aplicativo que os usuários interagem diretamente. É a interface visível e interativa que permite aos usuários interagir com o software ou site. O front-end é responsável por exibir informações, elementos gráficos e conteúdo visual aos usuários, bem como receber e processar suas entradas, como cliques do mouse, toques na tela e digitação.

Banco de dados

é um sistema organizado de armazenamento de informações, estruturado de forma a permitir o armazenamento, recuperação, gerenciamento e análise de dados de maneira eficiente. Ele é composto por tabelas que contêm registros de dados relacionados e é utilizado para armazenar informações relevantes para uma variedade de aplicações e finalidades.

Ciências das computação

um campo de estudo que se concentra na teoria, projeto, desenvolvimento e aplicação de sistemas de computação e software. Envolve a análise de algoritmos, estruturas de dados, teoria da computação e programação, bem como questões relacionadas à computação, como inteligência artificial, segurança cibernética, redes de computadores e banco de dados.
